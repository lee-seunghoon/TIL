## XPath란

> - HTML 태그에서 다양한 엘리먼트가 있고
> - 각 엘리먼트를 구분하기 어려울 경우가 있는데
> - 이 때, 어떤 엘리먼트를 지칭하는지 명확하게 하기 위해 사용하는 개념!

```tex
예시)
Q. 1반의 이강인을 부르고 싶은데... 동명이인이 있어서 애매할 때 XPath 사용

A. XPath_사용법 ='예시' : /학교/학년/반/학생[2] (학생 태그 중에서 2번째 값)
				'정식' : /html/body/div/span/a ...
				'요약' : //*[@학번="1-1-5"] == 모든 태그에서 학번이란 속성이 1-1-5인 값을 반환
                		//*[@id="login"]
                '실습' : /html/body/div[2]/div[3]/div[3]/div/div[2]/a == 네이버 로그인 버튼 Full
						//*[@id="account"]/a == 네이버 로그인 버튼 요약 XPath

ex) html tag

<학교 이름="나도고등학교">
	<학년 value="1학년">
		<반 value="1반">
			<학생 value="1번" 학번 value="1-1-1">손흥민</학생>
			<학생 value="2번" 학번 value="1-1-2">이강인</학생>
			<학생 value="3번" 학번 value="1-1-3">김민재</학생>
			<학생 value="4번" 학번 value="1-1-4">황의조</학생>
			<학생 value="5번" 학번 value="1-1-5">이강인</학생>
		</반>
		<반 value="2반"/>
		<반 value="3반"/>
		<반 value="4반"/>
	</학년>
```
