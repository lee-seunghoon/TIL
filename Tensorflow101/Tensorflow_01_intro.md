# Tensorflow 01 - 개요

- 텐서플로는 머신러닝 중 지도학습(**회귀, 분류**) 부분을 담당한다.
- 사용하는 알고리즘은 다음과 같다
  - `Decision Tree`
  - `Random Forest`
  - `KNN`
  - `SVM`
  - `Neural Network`

- 이 중에서 집중적으로 다룰 알고리즘 : `Neural Network`



## Neural Network

> 사람의 뇌 속에서 신경망 즉, 뉴런들의 활동을 모방한 알고리즘!

- 인공신경망 = Deep Learning 



## Process

- AI >> 머신러닝 >> 지도학습 >>  회귀,분류 >> 알고리즘 >> 라이브러리
